# Student Management System (JSP + JDBC)

A web-based Student Management System developed using JSP, Servlets, JDBC, and MySQL.

## Features
- Add student with roll number
- View all students (sorted by roll number)
- Edit student details
- Delete student with confirmation
- Responsive UI using CSS
- MySQL database integration

## Technologies Used
- Java
- JSP & Servlets
- JDBC
- MySQL
- Apache Tomcat
- HTML, CSS
- Eclipse IDE

## Project Structure
- `src/main/java` – Servlets & DB logic
- `src/main/webapp` – JSP pages & CSS
- `DBConnection` – Database utility class

## How to Run
1. Import project into Eclipse
2. Configure Apache Tomcat
3. Setup MySQL database
4. Update DB credentials
5. Run on server
